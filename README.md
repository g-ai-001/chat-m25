# chat-m25

仿微信安卓应用（本地单机版）

## 最新版本

### v0.10.0 (2026-03-29)

**MINOR版本 - 增加语音消息和位置分享功能，增强聊天体验**

- 新增语音消息功能：支持录制和发送语音消息，点击播放
- 新增位置分享功能：支持发送预设位置给好友
- 新增通知设置：支持启用/禁用通知、提示音、振动，可选择提示音类型
- 更新版本号至0.10.0

**MINOR版本 - 增加实用工具功能，增强用户体验**

- 新增闪光灯开关和相册选择按钮
- 设置页面增强：新增关于我们、用户协议入口
- 更新版本号至0.9.0

### v0.8.0 (2026-03-29)

**MINOR版本 - 扩展聊天和朋友圈功能，增强互动体验**

- 新增朋友圈评论功能：支持查看评论列表、发表评论
- 新增消息撤回功能：支持撤回已发送的消息

- 数据库升级至v7，新增comments表支持朋友圈评论

## 功能特性

- 仿微信UI设计，简洁美观
- 底部导航栏：微信、通讯录、我
- 聊天列表：显示聊天会话、未读消息数
- 聊天详情：支持发送和接收消息
- 聊天记录搜索：支持关键词搜索聊天内容
- 通讯录：管理联系人，支持星标朋友
- 个人中心：展示个人信息入口
- 朋友圈：浏览、发布、点赞朋友圈动态，支持评论
- 群聊管理：编辑群名称、设置群公告
- 表情包管理：分类浏览表情
- 数据备份：支持导出导入聊天记录
- 本地存储：所有数据保存在本地数据库
- 日志系统：便于问题排查和分析
- 消息撤回：支持撤回已发送的消息
- 消息已读状态：显示发送中/已发送/已送达/已读
- 扫一扫：支持扫描二维码和条码
- 语音消息：支持录制和发送语音消息，点击播放
- 位置分享：支持发送位置给好友
- 通知设置：支持自定义通知音效和振动

## 技术栈

- **语言**: Kotlin 2.3.20
- **目标SDK**: Android 16 (API 36)
- **构建工具**: Gradle 9.4.1 (Kotlin DSL)
- **UI框架**: Jetpack Compose + Material 3
- **架构**: MVVM + Clean Architecture
- **依赖注入**: Hilt
- **本地存储**: Room + DataStore
- **图片加载**: Coil
- **异步处理**: Kotlin Coroutines + Flow
- **导航**: Navigation Compose

## 项目结构

```
app/src/main/java/app/chat_m25/
├── data/               # 数据层
│   ├── local/         # 本地存储
│   │   ├── dao/       # Data Access Object
│   │   └── entity/     # 数据库实体
│   └── repository/     # 仓库
├── di/                 # 依赖注入模块
├── domain/             # 领域层
│   └── model/          # 领域模型
└── ui/                 # UI层
    ├── components/     # 公共组件
    ├── screens/        # 页面
    │   ├── home/      # 首页
    │   ├── contacts/   # 通讯录
    │   ├── emoticon/   # 表情包
    │   ├── favorites/  # 收藏
    │   ├── group/      # 群聊
    │   ├── profile/    # 个人中心
    │   ├── chat/      # 聊天详情
    │   └── moments/    # 朋友圈
    └── theme/          # 主题样式
```
