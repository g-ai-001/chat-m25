## 技术栈
- **语言**: Kotlin 2.3.20
- **构建工具**: Gradle 9.4.1 (Kotlin DSL)
- **UI框架**: Jetpack Compose + Material 3
- **架构**: MVVM + Clean Architecture
- **依赖注入**: Hilt
- **本地存储**: Room + DataStore
- **图片加载**: Coil
- **异步处理**: Kotlin Coroutines + Flow
- **导航**: Navigation Compose

## 设计规范
- 界面布局紧凑（修改Material 3 theme）
- 支持手机+折叠屏+平板自适应布局
- 支持120Hz以上刷新率
- 仅支持简体中文
- 禁止联网，纯本地单机使用
- 标题栏按钮不超过3个
- 搜索栏合并到标题栏

## 长期愿景
- 持续迭代优化用户体验
- 保持代码精简（<10000行）
- 定期进行代码重构优化
